1. **Set Up a Virtual Environment**

   ```bash
   python -m venv .venv
   ```

2. **Activate the Virtual Environment**

   - **Windows**:
     ```bash
     .venv\Scripts\activate
     ```
   - **macOS/Linux**:
     ```bash
     source .venv/bin/activate
     ```

3. **Install Dependencies**

   ```bash
   pip install -r requirements.txt
   ```

4. **Start server**

      ```
      python app.py
   ```
